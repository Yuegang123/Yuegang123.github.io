〇、基础概念

  • 数据（DATA）

    • 描述客户事物及其状态的符号记录（最原始的记录，未做加工）

  • 数据库（DATABASE）DB

    • 长期储存在计算机内，有组织的，可共享的大量数据的集合
    • 无冗余性：现实最多有一份数据与之对应

  • 数据库管理系统（DATABASE MANAGEMENT SYSTEM）DBMS

    • 一种计算机应用软件程序，用于创建和管理数据；与用户、其他应用程序及数据库进行交互以获取，管理和分析数据
    • 有故障恢复机制

  • 数据库系统（DATABASE SYSTEM）DBS

    • 包括5个部分
      • 1.数据库
        • 数据集合
      • 2.数据库管理系统（核心）
        • 负责数据库的规划，设计，协调，维护和管理
      • 3.应用程序
      • 4.数据库管理员
      • 5.用户
        • 应用程序开发人员
          • 程序猿，编写程序通过DBMS发出SQL请求
        • 最终用户
          • 通过程序访问数据库

  • T-SQL（transact structured query langage）【结构化查询语言增强版】

    • 流程控制、存储过程、触发器、游标、函数
    • T-SQL服务器端编程：提高代码可重用性、减轻客户端的负担、减少网络负担
    • 全局/局部变量定义：DECLARE @@变量名 数据类型（长度）/DECLARE @变量名 数据类型（长度）;
    • 赋初值：SET @变量名 = 表达式;
    • 不能同时对多个变量进行赋值；

  • 三范式

    • 1.列不可拆分
    • 2.唯一标识
    • 3.引用主键

三、sql语法

  • 1、单表查询

    • Order by: 按一个或多个属性列排序，ASC升序，DESC降序，默认升序

      • ```sql
        Select * from student order by sdept,sbirthday DESC –按系号升序排，同系的按生日降序排

    • Distinct:取消重复的行

      • ```sql
        Select distinct sno from score
      • ```sql
        字符类型：select sname+ssex as name_sex from student –王小二男
      • ```sql
        Right/left(‘张三丰’，1) 返回 “张” –返回字符串最右/左第N个字符
        • ```sql
          sqrt(2)返回 1.4142135623731 –返回X的平方根

    • 聚集函数

      • ```sql
        count（）
        Select count(        ) from student –查询学生总人数
      • ```sql
        Avg(*)
        Select avg(grade) from score –计算学生所有课程的平均成绩

    • Where条件查询

      • ```sql
        Between…and…
        Select sname from student where sage not between 20 and 25; –查询不是20到25岁之间的学生姓名
      • ```sql
        Like
        %:代表任意长度的字符串
        Select sname,sno from student where sname not like ‘刘%’ –查询不是姓刘的学生的姓名和学号
        _:代表任意单个字符
        Select sname,sno from student where sname like ‘_阳%’ –查询名字第二个字是阳的学生的姓名和学号
      • ```sql
        Not>and>or

    • Having

      • ```sql
        Select s_id,sum(s_score) from score group by s_id having sum(s_score) >= 200 –查询所有课程的总分超过200分的学生
      • ```sql
        Select a.sno,a.sname from student a, score b where a.sno = b.sno and b.cno = ‘2’ and b.grade > 90 –查询选修2号课程且分数大于90的学生学号和姓名

    • 外连接

      • ```sql

        表 Left join 表 on –列出左边关系中的所有元组

  • 4、子查询

    • 集合查询
      • 并（union）
        • ```sql
          Select sno from score where cno = 2 union all (select sno from score where cno = 3) –查询选修了课程号为2和3的学号 （没有all则去掉重复内容）
      • 差（except）【在第一个结果集中，但是不在第二个结果集】
        • ```sql
          （select * from student where sdept !=’CS’）except select * from student where sex =’男’ –查询非计算机系的非男生信息
      • 带比较的
        • § >, < =
      • 带any 或 all 的嵌套查询
        • § <, > ,<> –不等于子查询中的任何一个值/某个值
